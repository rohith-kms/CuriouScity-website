                UPDATING YOUR COPY
---------------------------------------------------

To update the copy on your computer to match the current development stage of the website, do the following:

1. Open "Terminal" from the dock at the bottom of your screen.
2. You can use the command "ls" to list all the folders
3. One of the folders is this one - "CuriouScity-website"
4. Type "cd CuriouScity-website" in the terminal to go into this folder
5. Type "git pull" in the terminal to fetch all the new folders from my GitHub account.
6. Your copy has been updated! Click on index.html to view the webpage.
